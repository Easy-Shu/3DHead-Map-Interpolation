# 3D Head-and-Neck Pressure Sensitivity

Two questions are proposed and solved in this paper:

1. Which 3D mesh interpolation method enables accurate computation of continuous 3D head PDT/PPT maps based on a sparse set of discrete anatomical landmarks?
2. Which data integration method can be effectively used to synthesize 3D head PDT/PPT maps from multiple studies into a consolidated, comprehensive outcome?
